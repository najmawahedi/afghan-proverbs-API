### How to run the application

### Clone the repository

git clone https://github.com/najmawahedi/afghan-proverbs-API
cd afghan-proverbs-app

### install the dependencies

npm install

### run the backend locally

cd back_end
npm install
npm start

### run the front end

cd front_end
npm install
npm start
